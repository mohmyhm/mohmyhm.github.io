# Mohammad Yahyaei's Personal Website

This is my personal website built using HTML and CSS, hosted on GitHub Pages.

## About

This website showcases my academic and professional background, including my research projects and publications.

## Deployment

This repository is configured for GitHub Pages. The website is automatically deployed from the `main` branch.

To update the website:
1. Make changes to `index.html` or other files.
2. Commit and push to the `main` branch.
3. GitHub Pages will automatically rebuild and deploy the site.

## Local Development

To view the website locally:
1. Clone the repository.
2. Open `index.html` in a web browser.
