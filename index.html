About

I am a Ph.D. candidate in Computer Science at Cleveland State University, specializing in Natural Language Processing (NLP), AI Safety and Secure Large Language Models, Agentic AI, and Interpretable and Efficient Language Models. My research focuses on developing advanced AI systems for cybersecurity, including encrypted traffic analysis, cryptographic API misuse detection, and secure code generation.

Research Projects

  • Transformer-Based Encrypted Traffic Intrusion Detection: Designed and implemented a framework using fine-tuned large language models for detecting malicious encrypted traffic through hexadecimal analysis.
  • Graph-Driven Transformer for Cryptographic API Misuse: Developed a model that integrates tokenized code sequences with Code Property Graphs to detect and classify Python cryptographic API misuse.
  • LLM-Based Secure Code Feedback Generator: Created a system using Retrieval-Augmented Generation (RAG), knowledge-graph reasoning, and vulnerability pattern evaluation for secure code feedback.
